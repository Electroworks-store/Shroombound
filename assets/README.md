# Asset Folders

These folders will contain game assets:

## /sprites
- `player.png` - Player spritesheet
- `enemies.png` - Enemy sprites
- `effects.png` - Slash, dust, particles

## /tilemaps
- `tutorial.json` - Room 1 Tiled map
- `arena.json` - Room 2 Tiled map
- `boss.json` - Room 3 Tiled map
- `tileset.png` - Shared tileset image

## /audio
- `slash.wav` - Attack sound
- `dash.wav` - Dash sound
- `jump.wav` - Jump sound
- `hurt.wav` - Damage sound
- `enemy_hit.wav` - Enemy damage
- `boss_theme.mp3` - Boss music

---

**Note:** For the MVP, placeholder graphics are generated programmatically in `BootScene.ts`. 
Add real assets here when ready for polish phase.
